Configure and manage all aspects of your AI Agent application — from its profile and language support to developer tools, variables, and advanced settings.

## App profile

Configure your app's name, description, icon, target audience, language defaults, subdomain, and custom HTTP error messages.

## Language management

Build and manage multilingual AI Agents with support for multiple languages, translation services, and language-specific NLU models.

| Topic | Description |
|---|---|
| **Get Started** | Overview of multilingual AI Agent capabilities |
| **Supported Components & Features** | Language-specific component and feature compatibility |
| **Manage Languages** | Add, update, and configure languages |
| **Manage Translation Services** | Configure Microsoft Translator, Google Translation, or custom engines |
| **Multilingual AI Agent Behavior** | Language-dependent NLP behaviors and feature support |

## Dev tools

Access developer tools for building, securing, and integrating your AI Agent — including custom scripts, authorization profiles, SDK setup, and API scopes.

| Topic | Description |
|---|---|
| **App Functions** | Create and manage reusable JavaScript utility functions |
| **Authorization Profiles** | Overview of supported authentication models |
| **Web/Mobile SDK** | Set up and integrate the Kore.ai Web SDK |
| **Mock Scenes** | Prototype and preview conversations using Conversation Designer |

### Authorization profiles

| Auth type | Description |
|---|---|
| **Basic Auth** | Configure username/password authentication |
| **OAuth v1** | Set up OAuth v1 with consumer key and secret |
| **OAuth v2** | Set up standard OAuth v2 authorization |
| **OAuth v2 Password Grant** | Configure OAuth v2 password grant authorization |
| **OAuth v2 Client Credentials** | Configure OAuth v2 client credentials flow |
| **Kerberos SPNEGO** | Set up Kerberos SPNEGO for enterprise authentication |

## Using variables

Define environment and content variables to manage configuration values and reuse them across your app's tasks, nodes, and responses.

| Topic | Description |
|---|---|
| **Using Variables** | Create, manage, and use environment and content variables |
| **Collections** | Store and manage multiple value sets for environment variables |

## Managing namespaces

Organize variables into namespaces and map them to app components for optimized loading during execution.

## Advanced settings

Fine-tune session behavior, data privacy, URL management, and system-level configurations.

| Topic | Description |
|---|---|
| **PII Settings** | Redact, replace, or mask personally identifiable information |
| **Manage Sessions** | Configure conversation session duration and closure behavior |
| **Manage URLs** | Set expiry and access limits for attachment and chat history URLs |
| **System Settings** | Configure call recording, data retention, and other system options |

## Change logs

Review a complete history of modifications made to your app across all modules. View, filter, and export changes across Automation AI, Search AI, Agent AI, and more.

## Delete app

Permanently remove an app, with an optional export and a 30-day recovery window for published apps.

<Warning>App deletion is permanent for unpublished apps. Published apps have a 30-day recovery window. Export your app definition before deleting.</Warning>
